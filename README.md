About this submission
This is the front-end prototype for our proposed capstone project, a web-based system that helps schools record, search, claim, and manage lost-and-found items through a centralized MySQL database instead of paper logbooks and word-of-mouth.

The prototype is a static HTML/CSS/JavaScript site that demonstrates the look, navigation, and intended user flow of the final application. It is not connected to a real database — forms and buttons simulate what the finished system will do (e.g. login "succeeds" for any input, the records table filters a hard-coded list, the report generator randomizes sample numbers).
